  • Help
    • What is BubbleMapper?
      • BubbleMapper is a mind mapping tool. A mind mapper is a tool used to link thoughts and ideas together to indicate a relationship. Each bubble is a thought, and each line a relationship.

        Mind mapping can be used for almost any task normal notes would be used for, and often a mind map is a lot more intuitive when reading them.
    • Getting Started
      • To create a bubble, double click anywhere on the map.

        Once created, you should see a blinking caret in the bubble. Typing there will allow you to add and remove text from the bubble. Clicking away from the bubble finalizes the text, preventing accidental change in the future. Don't worry, you can change it later!

        To establish some relationships, you must connect a bubble to another bubble. To do so, click the connect button above the first bubble and then dragged the dashed line to the second bubble. Clicking on the second bubble should connect it.

        That's the basics! Have fun mind mapping!
